This is the "historic crater lake" test in REACT.
You can use a browser plugin to solve the CORS problem.

### Installation

install the dependencies 

```
npm install
```
or
```
yarn install
```

### Run

```
npm run dev
```
or
```
yarn dev
```

### Build 

`/build` directory will be created on the root app directory

```
npm run build
```
or
```
yarn build
```
